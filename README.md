# ProdLife

ProdLife is an all-in-one daily productivity system for Obsidian. It creates daily notes from day-specific templates, rolls unfinished tasks forward, schedules recurring work, manages reminders, and turns completed tasks into streaks, achievements, and a GitHub-style activity heatmap.

> ProdLife 1.0 is under active development. Installation and usage documentation will be completed with the release build.

## Development

```bash
npm install
npm run check
```

Copy `main.js`, `manifest.json`, and `styles.css` into `<vault>/.obsidian/plugins/prodlife/` to test locally.
